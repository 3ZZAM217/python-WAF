# Security Policy

## Supported Versions

Currently, only the latest `main` branch is receiving security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within Python Shield WAF, please do **NOT** open a public issue.

Instead, please send an email to the repository maintainers. We will respond within 48 hours to acknowledge the report and outline the next steps for mitigation.

When reporting, please include:
* Description of the vulnerability
* Steps to reproduce
* Potential impact

Thank you for helping keep Python Shield WAF secure!
